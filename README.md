[Aseprite](https://github.com/aseprite/aseprite) binary build for 64-bit Linux,idea inspired from [this repo](https://github.com/mmozeiko/aseprite-bin?tab=readme-ov-file)
Instructions are same:
1.Fork Repository
2.Go to Actions
3. Open aseprite workflow, and click Run workflow
4. Wait ~13min for build to finish, then open latest run
5. Scroll to the bottom to download .zip archive

For building newer aseprite version repeat steps 3 to 5.
